AniJS

Declarative animation library for the web.

Fork on GitHub

Some Cool Demo

Click Square Demo

When click, Where Square, What Container Boxes, How flipInY animation 

                            <div class="demo-square demo1" data-anijs="when: click, what: .container-box, how: flipInY" ></div>

Simple Stacking Animation

When focus, Where name input What paragraphs, How wobble animation

When animationend, Where This paragraph What pre tag below How hinge animation 

                            <input id="name" type="text" data-anijs="when: focus, what: p, how: wobble">

<pre data-anijs="when: animationend, where: p.p-demo2, how:hinge">
